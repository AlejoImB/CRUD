Bienvenido a la API de Tareas

Puedes usar Postman para interactuar con los siguientes endpoints:

  • GET - Obtener todas las tareas: /tareas/
  • POST - Crear una nueva tarea: /tareas/
  • GET - Obtener una tarea por ID: /tareas/{tarea_id}
  • PUT - Actualizar una tarea: /tareas/{tarea_id}
  • DELETE - Eliminar una tarea: /tareas/{tarea_id}
